天井照明（調光調色可能）


## Description

ユーザは，拡張現実を用いることで直感的に照明の制御ができる．  
今回はHMDとしてOculus Rift，ステレオカメラとしてOvervisionを用いている．  
  
拡張現実として表示するUIはユーザの目の前に表示される．  
UI上のボタンを注視することで，選択することができる．
この際，Unityを用いており，主にC#でプログラムの実装を行っている．

##Equipment

* HMD(Head Mount Display)  
* モーションセンサ  
* コンピュータ  

## Usage

1. 機器を装着
2. UI上の任意のボタンを注視
3. 照明の明るさ，色合い等が変更
